# KOA-AJAX-PARAMS

The Koa Middleware used to auto parse get & post request params.

## Install

```bash
npm i koa-ajax-params
```

```bash
yarn add kao-ajax-params
```

## Usage

```javascript
import Koa from 'koa';
import router from 'koa-router';
import ajaxParams from 'koa-ajax-params';

const app = new Koa();
const route = new router();
```
